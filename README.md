# 🤖 AI SQL Assistant

Convert natural language into SQL queries using AI.

## 🚀 Features
- Natural Language → SQL
- Voice Input 🎤
- Chat UI 💬
- SQLite Database
- Backend Security 🔐

## 🛠 Tech Stack
- Python
- FastAPI
- LangChain
- Ollama (LLaMA3)
- HTML, CSS, JavaScript

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Start backend:
uvicorn main:app --reload

3. Open frontend:
Open index.html in browser
